Project

A miniproject (Project A) is performed which involves implementing a state machine and performing thorough analysis of its design and performance. Part 2 concerns digital system design, focusing on HDL coding and applying these skills in lab practicals to simulation state machines and other constructs that are useful in embedded system design.

Students taking the course in a Computer Science programme as required to complete an addition project, or more ideally, larger project that builds upon Project A as a starting point to extent it further; this project is altogether approximately 4 credits of effort, and a significant focus of the computer science project is to incorporating network programming (i.e. prototyping a IoT type sensor) in addition to thorough modelling using for instance more detailed UML and statemachines.
